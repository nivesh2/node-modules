# Documentation

## About
Basic API Server using Express framework and Node.JS.

**Middleware** used are: 
 * body-parser *for parsing request body*
 * morgan *for logging every request*
 


### Run: 
```shell 
  DEBUG='main:*' node app.js
```  

### Run with nodemon:
```shell 
  DEBUG='main:*' nodemon node app.js
```
